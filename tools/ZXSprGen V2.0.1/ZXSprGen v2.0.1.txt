ZX Pascal Sprite generator v2.0.1 - генератор спрайтов для ZX Like Pascal.

На вход подаётся картинка Windows Bitmap (.bmp) с чёрно-белым изображением.

Опции:
- "Спрайты с атрибутами." включает в спрайты коды атрибутов из поля "Список атрибутов".
  Если указано недостаточное количество атрибутов или не указано вообще, то остальные
  спрайты получают атрибут 56, соответствующий чёрным чернилам на белой бумаге.
  Спрайт состоит из блоков 8х8 точек. Картинка разбивается на блоки слева направо, затем сверху вниз.
- "Генерировать шестнадцатеричные числа (16h)." включает генерацию чисел атрибута и спрайта в шестнадцатеричном виде вместо десятичного.
- "Префикс числа" содержимое этого поля вставляется перед числом в шестнадцатеричной форме.
  Для инициализации массива Pascal требуется символ "$", для ассемблера требуется символ "#".
- "Вставлять префикс строки" вставляет в начало каждой строки подстроку из поля. Требуется для задания массива для ассемблера.
- "Запятая в конце строки" заканчивает каждую строку запятой. Требуется для инициализации массива Pascal.
- "X", "Y", "Ширина", "Высота" задают окно спрайта. Позволяет создать несколько спрайтов из одного изображения.
  Если "Ширина" или "Высота" равны 0, то в спрайт преобразуется всё изображение. Заданый спрайт отображается графически на картинке.

Сохранение результатов не предусмотрено. Их можно только скопировать.
